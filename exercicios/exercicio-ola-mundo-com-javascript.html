Olá Mundo! - Javascript

Maio 28, 2021 by Boaventura M.

Como é o costume de o primeiro programa de um novato em programação é imprimir na tela o famoso "olá mundo", aqui não faremos  diferente.

Javascript tem várias maneira de imprimir dados no browser:

  • Podemos utilizar o innerHTML para imprimir dados em  elemento HTML,
  • Podemos utilizar o document.write() para escrever dentro de um documento HTML,
  • Podemos utilizar window.alert() para escrever na caixa de alerta do browser,
  • Ou ainda podemos utilizar console.log() para imprimir no console do browser.

Considera o código: (click aqui para testar)

<!DOCTYPE html>
<html>
<body>

<p id="demo"></p>

<script>

document.getElementById("demo").  innerHTML = "Olá Mundo!";
</script>

</body>

</html>

            Saída no navegador: Olá Mundo!

 

Explicar o código:

Para acessar um elemento HTML o javascript tem vários métodos, nesse caso usamos document.getElementById("id"). Use document.getElementById("id")  para acessar os  elementos exclusivamenter através do id (ById).

O atributo id define qual elemento será acessado, no nosso caso temos um elemento que possuí o id especificado no metódo document.getElementById("demo"), que é o id="demo"que está no paragráfo <p id="demo"></p>.

A propriedade innerHTML do javascript define qual conteúdo será imprimido no browser, nesse caso definimos a palavra (innerHTML = "Olá Mundo") Olá mundo.

A propriedade innerHTML é a mais utilizada para definir conteúdo aos elemento HTML.

 

Usando document.write()

Tenha muita atenção em usar document-write() para imprimir dados no browser. Se utilizarmos o document.write() depois da página for carregada deletará todo HTML existente no documento.

Use o document.write() apenas  para proposito de teste.

<!DOCTYPE html>
<html>
<body>

<p id="demo">Olá mundo, será imprimido por baixo</p>

<script>

document.write("Olá Mundo");
</script>

</body>

</html>

            Saida no navegador:

                    Olá mundo, será imprimido por baixo

                    Olá Mundo!

 

Utilizar o window.alert():  Click aqui

Utilize o windows window.alert() para escrever na caixa de alerta do browser. o metódo de uso é semalhante do document.write(). ex: (window.alert("olá mundo")) ou simplesmente (alert("olá mundo")).

Usando console.log():

Use o console.log() para fins de depuração.

 

Considerações:

Com este exercício simples você aprendeu algumas maneiras de imprimir dados com javascript, que com certeza veremos mais as suas aplicações em exercícios futuros.
